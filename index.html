Kompjuterske mreže
Kompjuterska mreža je grupa povezanih uređaja koji razmjenjuju podatke i dijele resurse, npr. internet, printer ili datoteke.

Kviz, IP adrese, Command Prompt komande, topologije i slanje poruka kroz mrežu.

IPv4 intervali lokalnih adresa

10.0.0.0 - 10.255.255.255
172.16.0.0 - 172.31.255.255
192.168.0.0 - 192.168.255.255

Za ruter se u lokalnoj mreži najčešće rezerviše IP adresa iz istog intervala kao i ostali uređaji, a obično završava sa .1, npr. 192.168.1.1.

Struktura IPv4 adrese

IPv4 adresa ima 4 okteta od 0 do 255. Ovo je manji podsjetnik uz primjer adrese.

✅ Kviz

IP adrese: razvrstaj i dopuni

Prvo ide 6 IP adresa za razvrstavanje. Zatim slijede 3 mrežna crteža u kojima na osnovu jedne poznate adrese upisuješ validne IP adrese za ostale uređaje. Na kraju dobijaš još 1 veći zadatak za cijelu lokalnu mrežu.

💻 CMD i pojmovi

Kako pronaći vlastitu IP adresu

Na kompjuteru otvori Command Prompt, upiši ipconfig, a zatim pronađi IPv4 Address i Default Gateway.

KorakŠta radiš
1Pretraži cmd.
2Otvori Command Prompt.
3Upiši ipconfig.
4Pronađi IPv4 Address i Default Gateway.
5Komandom ping 8.8.8.8 provjeri konekciju.
IPv4 Address

Adresa tvog uređaja u lokalnoj mreži.

Default Gateway

Najčešće adresa rutera. U ovoj lekciji to je obično lokalna adresa koja završava na .1.

Globalna IP

Adresa kojom se mreža predstavlja prema internetu.

IPv6

Novija verzija IP adrese sa dužim zapisom; ovdje vježbamo IPv4.

Iz prikaza komande ipconfig prepiši svoju IPv4 adresu i adresu rutera (Default Gateway).
